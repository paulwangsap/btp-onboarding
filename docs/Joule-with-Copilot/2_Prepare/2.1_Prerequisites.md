# Prerequisites

* You have at least one SAP business tenant (Such as SAP S/4HANA Private Cloud, SAP SuccessFactors) integrated with **Joule**.
* You have set up trust between Entra and [SAP Cloud Identity Services](https://help.sap.com/docs/cloud-identity-services "https://help.sap.com/docs/cloud-identity-services") – Identity Authentication, registering Entra as the Corporate IdP based on OIDC. For more information, see [Configuring SAP Cloud Identity Services and Microsoft Entra ID for Joule![Information published on SAP site](https://help.sap.com/doc/6189c8655c484916bb8eb767126a653a/CLOUD/en-US/themes/sap-light/img/sap_link.png "Information published on SAP site")](https://help.sap.com/docs/link-disclaimer?site=https%3A%2F%2Fcommunity.sap.com%2Ft5%2Ftechnology-blog-posts-by-sap%2Fconfiguring-sap-cloud-identity-services-and-microsoft-entra-id-for-joule%2Fba-p%2F14105743 "https://community.sap.com/t5/technology-blog-posts-by-sap/configuring-sap-cloud-identity-services-and-microsoft-entra-id-for-joule/ba-p/14105743").
* All users that shall be able to use **Joule** from within Microsoft 365 Copilot have been created / replicated to the SAP Cloud Identity Services User Store.
* You have a Microsoft M365 Copilot license and have rolled out M365 Copilot to the intended user base. For more information, see [Getting Started with Microsoft 365 Copilot![Information published on non-SAP site](https://help.sap.com/doc/6189c8655c484916bb8eb767126a653a/CLOUD/en-US/themes/sap-light/img/3rd_link.png "Information published on non-SAP site")](https://help.sap.com/docs/link-disclaimer?site=https%3A%2F%2Flearn.microsoft.com%2Fen-us%2Fcopilot%2Fmicrosoft-365%2Fmicrosoft-365-copilot-setup "https://learn.microsoft.com/en-us/copilot/microsoft-365/microsoft-365-copilot-setup").
