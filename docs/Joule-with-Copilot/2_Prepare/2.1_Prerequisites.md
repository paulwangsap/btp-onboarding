* You have set up trust between Entra and [SAP Cloud Identity Services](https://help.sap.com/docs/cloud-identity-services "https://help.sap.com/docs/cloud-identity-services") â€“ Identity Authentication, registering Entra as the Corporate IdP based on OIDC.
